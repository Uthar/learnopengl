# Description

learnopengl.com tutorials done in Allegro and Common Lisp.

# Requirements

OpenGL 3.0 compatible driver

# Run

With Nix:
```
nix-shell
sbcl --script run.lisp
```

# Movement

- ESC: capture mouse
- Mouse: look around - press ESC first
- WASD: movement
- Space: move up
- V: move down
- F: toggle flashlight
